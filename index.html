Experiência

PePlay Diversões - Tecnico de Informatica - Desenvolvedor
  • Criado um sistema em Java de caixa e gestão de estoque.

  • O modulo do caixa consiste em armazenar os pedidos dos clientes e fazer a soma do valor dos pedidos.
    Quando o cliente for fechar a conta ele imprime o cupom fiscal e apos o pagamento, armazena o valor da conta, faz a subtração do estoque e manda um e-mail para o dono com o valor da conta.

  • O modulo de fechamento de caixa soma os valores das contas pagas do dia e mostra o valor final do caixa.
    Após o fechamento, ele armazena o valor do caixa e dia para controle de dados e manda uma planilha no excel com todos os dados do dia por email.

  • O modulo de estoque, faz o cadastro de fornecedores, produtos e dos produtos a serem vendidos, e automaticamente junto com o modulo do caixa faz a subtração do estoque conforme acontece a venda desses produtos.
    Todos os dados estão disponíveis para alteração e visualização para usuários com autorização de administrador (Login e Senha)

  • O Sistema foi feito utilizando Maven para as APIs e JDBC para banco de Dados
